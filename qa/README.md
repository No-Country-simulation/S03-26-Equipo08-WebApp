# 🧪 QA & Testing - Camila

¡Bienvenida, Camila! Esta carpeta es tu espacio para organizar todo lo relacionado con el aseguramiento de la calidad (QA) del proyecto.

## 📂 Estructura sugerida

Puedes organizar tus archivos de la siguiente manera:

- `documentacion/`: Planes de prueba, casos de prueba y reportes.
- `automatizacion/`: Scripts de pruebas (Cypress, Playwright, Selenium, etc.).
- `evidencias/`: Capturas de pantalla o videos de errores encontrados.
- `bugs/`: Reportes detallados de errores (si no usamos Jira/GitHub Issues).

## 🚀 Cómo subir tus cosas

1. Coloca tus archivos en las subcarpetas correspondientes.
2. Desde la terminal en la raíz del proyecto:
   ```bash
   git add qa/
   git commit -m "qa: add [descripción de lo que subes]"
   git push origin main
   ```

_Si necesitas ayuda con Git o quieres cambiar la estructura, no dudes en preguntar._
