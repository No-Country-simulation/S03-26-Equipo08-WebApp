# Frontend de S03-26-Equipo08-WebApp

Este es el directorio que contiene el código fuente de la interfaz de usuario de nuestro proyecto.

## Tecnologías Utilizadas

- **React** (con Vite)
- **Vanilla CSS** para el diseño
- **React Router** para la navegación
- **Fetch/Axios** para las llamadas a la API

## Diseño (Figma)

- [Acceder al diseño en Figma](https://www.figma.com/make/7pSM0BXRpccYHLbzLF5Iz5/CMS-para-testimonios?t=Gv5JJPkex9eRHR4c-0)

## Configuración y Ejecución

1. Instalar las dependencias con `npm install`.
2. Iniciar el servidor de desarrollo con `npm run dev`.
3. El frontend estará disponible en `http://localhost:5173`.

## Diseño y Estilo

Se prioriza una estética premium, moderna y responsive, alineada con la identidad visual del proyecto.

## Integración con el Backend

La conexión se realiza a través de las rutas expuestas por el servidor del backend, gestionando la persistencia de los tokens JWT en el almacenamiento local.
